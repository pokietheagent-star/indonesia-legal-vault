# PRECENT_INDEX

This file is the repository’s issue-to-authority cross-reference for Indonesian legal analysis.

Purpose:
- link legal issues to controlling statutes,
- link legal issues to relevant Mahkamah Agung (MA) and Mahkamah Konstitusi (MK) decisions,
- preserve citation traceability,
- and help Hakim and Researcher avoid re-solving settled questions from scratch.

## Usage rules

1. Start from the legal issue, not from the case name.
2. Identify the governing statute or regulation first through pasal-id MCP.
3. Add the relevant precedent only after the statute anchor is confirmed.
4. Keep the index issue-based so it can be used for triage, drafting, and synthesis.
5. Mark any authority as verified, needs review, or provisional.

## Suggested index columns

- Issue category
- Sub-issue
- Governing statute / regulation
- Relevant pasal(s)
- MA decision(s)
- MK decision(s)
- Holding summary
- Current status
- Freshness date
- Confidence level
- Notes for Hakim

## Example entry format

- Issue: unlawful act / tort
- Statute: KUHPerdata Pasal 1365, Pasal 1320, Pasal 1338
- MA: [decision number]
- MK: [decision number if relevant]
- Summary: [short holding]
- Status: verified / needs review / provisional
- Notes: [litigation or drafting relevance]

## Maintenance rule

This index is only useful if it is updated when laws change, cases are superseded, or the legal theory in a memo evolves.
